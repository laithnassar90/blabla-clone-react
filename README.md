## Frontend for Wassel ridesharing application using React + Redux + Rails API

#### Description
Wassel ridesharing application which uses React + Redux + Rails API.

## Setup
Install packages:
```
$ npm install/yarn install
```

## Usage
Run server:
```
$ npm start/yarn start
```
